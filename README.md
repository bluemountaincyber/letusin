# LetUsIn

## Overview

This repository contains the code and instructions for accessing your resources in a variety of environments. Follow the instructions in each folder to prepare for an audit by Blue Mountain Cyber.

## Environments Covered

- Amazon Web Services (AWS)
- Kubernetes

## Prerequisites

Clone this repository to a Linux or macOS system/container/Cloud Shell with the following tools installed:

### Kubernetes

- [kubectl](https://kubernetes.io/docs/tasks/tools/)

### AWS

- [AWS CLI](https://aws.amazon.com/cli/)
- [Terraform](https://www.terraform.io/downloads.html)
